## How I bulit the weather App:
The final product of this project demonstrates the practical application of several concepts learned during my three-month program at ALT School Africa. Key concepts implemented include async/await for API consumption, Sets for maintaining unique values, Local Storage for data persistence, unshift() for adding recent searches to the beginning of the search history, slice() for limiting the display to the five most recent searches, and functions for processing data and dynamically rendering content to the DOM. These concepts collectively contributed to the functionality and user experience of the application.
